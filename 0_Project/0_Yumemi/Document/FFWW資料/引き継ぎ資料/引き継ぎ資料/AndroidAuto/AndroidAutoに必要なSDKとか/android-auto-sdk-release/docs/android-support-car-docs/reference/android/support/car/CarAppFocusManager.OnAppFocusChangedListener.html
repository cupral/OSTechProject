public abstract void onAppFocusChanged (CarAppFocusManager manager, int appType, boolean active)

Indicates the application focus has changed. The CarAppFocusManager instance causing the change does not get this notification.

Parameters
manager CarAppFocusManager: the CarAppFocusManager this listener is attached to. Useful if the app wished to unregister the listener.
appType int: application type for which status changed
active boolean: returns true if active
