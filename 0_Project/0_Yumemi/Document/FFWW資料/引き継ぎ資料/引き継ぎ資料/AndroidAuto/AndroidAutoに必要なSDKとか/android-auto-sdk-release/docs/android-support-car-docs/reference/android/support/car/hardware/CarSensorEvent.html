Class Overview

A CarSensorEvent object corresponds to a single sensor event coming from the car. Sensor data is stored in a sensor type-specific format in the object's float and byte arrays.

To aid in unmarshalling the object's data arrays, this class provides static nested classes and conversion methods (such as CarSensorEvent.DrivingStatusData and getDrivingStatusData()). Additionally, calling a conversion method on a CarSensorEvent object with an inappropriate type results in an UnsupportedOperationException being thrown.
