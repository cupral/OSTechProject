Class Overview

Provides a binding from the app-specific data set to the MenuItem displayed within the menu. The structure is similar to android.support.v7.widget.RecyclerView.Adapter.

To implement menu hierarchy, override onLoadSubmenu(int) and set the menu type to SUBMENU.

public void onEnter ()

Called when the user enters this menu. For example, when a user opens the drawer, goes into a submenu, or backs out of a previous submenu.

public void onExit ()

Called when the user exits this menu. For example, when a user goes into a submenu, presses back, or closes the drawer.

public MenuAdapter onLoadSubmenu (int position)

Called when the system wants to load a submenu adapter. Called only once unless the menu item is invalidated via notifyDataSetChanged() or notifyItemChanged(int).

Parameters
position int: The position of the item within the adapter's dataset
Returns
MenuAdapter A menu adapter that supplies the data for the requested submenu. Note: The menu adapter doesn't need to have all its data ready when it's returned from this function; the adapter can load its data inside onEnter() when it is actually displayed.
