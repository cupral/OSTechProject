Class Overview

Allows apps to add Android Auto extensions to notifications. To create a notification with this Extender:

  1. Create a NotificationCompat.Builder and set desired properties.
  2. Create a CarNotificationExtender.
  3. Set car-specific properties using the add and set methods of CarNotificationExtender.
  4. Call extend(NotificationCompat.Builder) to apply the extensions to a notification.
  5. Post the notification to the notification system with the NotificationManagerCompat.notify(...) methods (do not use the NotificationManager.notify(...) methods). 
 Notification notification = new NotificationCompat.Builder(context)
         ...
         .extend(new CarNotificationExtender.Builder()
                 .set*(...)
                 .build())
         .build();

To access car extensions on an existing notification, use the CarNotificationExtender(Notification) constructor then use get methods to access values.
