public interface

CarVendorExtensionManager

com.google.android.apps.auto.sdk.service.vec.CarVendorExtensionManager

Class Overview

This class is used to read from or write to vendor extensions.

An instance of CarVendorExtensionManagerLoader should be acquired from getCarManager(String) with a service name of VENDOR_EXTENSION_LOADER_SERVICE. Then users can request a specific manager via the getManager(String) method. The service name requested should be the service_name of the vendor extension channel as defined in the GAL protocol. Vendors are responsible for encoding and decoding their messages as the data[] variable is simply a pipe.

While an application is using a vendor extension service, no other applications can use the vendor extension service with the same service name.

An instance of CarVendorExtensionManager is not thread-safe.
